With this plugin admin can post the WordPress Post to remote sites (Different Wordpress site) from one single place.

== Description ==

Using Post Content XMLRPC plugin admin is allowed to post the content to remote sites from one single place. This plugin uses the cURL & XMLRPC service. This plugin adds the post in default category. You can add post tags while adding the post.

Plugin Requirements other than Wordpress :

1.Please verify Current site & remote sites has CURL enabled. 
2.Please verify remote site has XMLRPC enabled from Settings >> Writing.

This plugin extends the functionality/usability shown on :
http://www.wprecipes.com/post-on-your-wordpress-blog-using-php 


== Installation ==

Upload "Post Content XMLRPC" plugin to your blog, Activate it. That's it!!

e.g.

1. Upload the entire 'post-spoon-content' to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Post Content' menu in your WordPress admin panel.

== Frequently Asked Questions ==

= How do i add sites using this plugin & what is use of it? =

1. Find 'Post Content' menu in your WordPress admin panel. Click on "add sites" menu.
2. Add site title for your reference to identify site later on.
3. Add the url of remote site where you want to add the Wordpress Post.
4. You have to add correct username & password which is having atleast "add post" access.

All added site you will see listed when you try to add post to multiple sites using this plugin.

= Which category & tags will be assigned to post on remote blog? =

Post will be added to default category present on remote blog. You can add post tags while adding post. It will be created & assinged automatically if it's not already present on remote blog.

= Can i upload content having image/other attachments? =
No. Sorry you are only limited to add only online images/attachments in post content. If you uplaod new image/attachment in content, it will be hosted from current site only.
